# Тестовое задание (Random User)

## Описание проекта
Приложение разработано в рамках тестового задания для компании Workmate.
Основная цель – генерация случайных пользователей с использованием API [randomuser.me](https://randomuser.me/).

## Архитектура и подход
- MVVM + Clean Architecture
- Jetpack Compose для UI
- Dependency Injection (Hilt)
- Асинхронность: Kotlin Coroutines + StateFlow
- Retrofit + Kotlinx.serialization — работа с API  
- Работа с базой: RoomDB
- Navigation Component для перехода между экранами

## Библиотеки:
- Compose
- Navigation compose
- Hilt
- Coroutines
- Coil
- Room
- Retrofit
- Kotlinx.serialization.json
- OkHttp
